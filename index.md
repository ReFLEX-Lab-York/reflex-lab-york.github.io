## Getting Started

New to the lab? Complete these steps:
1. Request SSH access to the [Lab Server]({% link wiki/lab-server.md %}) and create a project workspace.
2. Review [Computing Resources]({% link wiki/computing-resources.md %}) to pick the right machine for your workload.
3. Follow the per-device guides (GPU server, Pi cluster, robots) before running long jobs.
4. Log your experiments and hardware bookings so teammates can coordinate usage.

For any questions or issues, please contact the lab administrator.
